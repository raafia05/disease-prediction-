# 🏥 Disease Prediction System

## 📌 Overview
This project predicts the likelihood of disease using Machine Learning.

## ⚙️ Features
- Random Forest Classifier
- Cross-validation
- Simple prediction system

## 📊 Accuracy
- Test Accuracy: ~90%
- Cross-validation Accuracy: ~85–95%
